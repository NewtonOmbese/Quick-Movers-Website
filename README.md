## Description

 <p>This project is a web application for Quick Movers that shows details about the services they offer, has information about the movers company, a work with us that simply contacts the company for moving.</p>

## Features

As a user of the application,you will be able to :

1. See a description of services offered by Quick Movers
2. Can reach out to inquire more about the company
3. Can contact and organise for moving with the company

 ###  Requirements

 * Access to  a computer or any other garget

 * Access to internet

### Installation Process

 ****  
* Clone to the repo : git clone https://github.com/NewtonOmbese/Quick-Movers-Website.git

* Unzip the downloaded files in a folder of choice.

* Open the index file from the zipped file with any browser.
